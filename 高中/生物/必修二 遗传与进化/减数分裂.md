# 减数分裂



## 1. 定义

> **【有丝分裂】**
>
> 作用：把一个细胞通过复制合成为两个子细胞，子细胞与亲本细胞一模一样。
>
> ![image-20230507212636081](https://jihulab.com/AmosHogg/picbed/-/raw/main/pictures/2023/05/7_21_26_36_202305072126148.png)

**定义：**减数分裂是进行有性生殖的生物，在产生成熟生殖细胞（配子）时进行的染色体数目减半的细胞分裂。在减数分裂前，染色体复制一次，而细胞在减数分裂过程中连续分裂两次。

## 2. 减数分裂过程

- 间期：DNA复制 蛋白质合成

- 减数第一次分裂

  ![image-20230507214213244](https://jihulab.com/AmosHogg/picbed/-/raw/main/pictures/2023/05/7_21_42_13_202305072142261.png)

  - 前期：***同源染色体***[^1]***联会***[^2]形成***四分体***[^3]；

    ​			**同源染色体**非姐妹染色单体***交叉互换***[^4]。

  - 中期：**同源染色体**排列在赤道板两侧。

  - 后期：**同源染色体**分离，非同源染色体自由组合。

  - 末期：形成两个子细胞

- 减数第二次分裂【与有丝分裂相似】

  ![image-20230507215729709](https://jihulab.com/AmosHogg/picbed/-/raw/main/pictures/2023/05/7_21_57_29_202305072157778.png)

  - 前：膜仁消失显两体
  - 中：形数清晰赤道齐
  - 后：姐妹分开均两极
  - 未：膜仁重现两体失

> [注意] 染色体、染色单体和姐妹染色单体的区别
>
> 1. 一个着丝粒就是一条染色体，包括两种情况：
>
>    1. 没复制的染色体不含染色单体。
>
>    2. 复制后的一条染色体含两条染色单体。没有独立着丝粒的都称为染色单体，共用着丝粒的两条染色单体称为姐妹染色单体。
>
> 2. 一条染色体复制后，形成两条，这两条通过着丝点连在一块，这两条未分开的就是单体，分开后，就是二条染色体。
>
> 3. 姐妹染色单体是由间期复制而来。

## 3. 精卵形成区别

| 比较项目       | 精子                  | 卵细胞                                                 |
| -------------- | --------------------- | ------------------------------------------------------ |
| 产生部位       | 睾丸或精巢            | 卵巢                                                   |
| 原始生殖细胞   | 精原细胞              | 卵原细胞                                               |
| 细胞质分裂情况 | 两次都均等分裂        | 只有减数第二次分裂中第一极体分裂均等，其他分裂皆不均等 |
| 分裂结果       | 1个精原细胞 → 4个精子 | 1个卵原细胞 → 1个卵细胞和3个极体                       |
| 变形阶段       | 有                    | 无                                                     |

# [题型] 基础概念

1. 同源染色体是指（）

   A. 一条染色体复制形成的两条染色体 → 非复制

   B. 减数分裂过程中联会的两条染色体 → ✅

   C. 形态特征大体相同的两条染色体 → 一个来自父方，一个来自母方

   D. 分别来自父亲和母亲的两条染色体 → 大小一般相同

2. 有关人类正常细胞中同源染色体的说法中错误的是（）

   A. 精细胞中不存在同源染色体 → ✅

   B. 次级精母细胞中存在同源染色体 → 减数II分姐妹，所以次级精母细胞中不存在同源染色体

   C. 体细胞中有23对同源染色体 → ✅

   D. X和Y染色体也是一对同源染色体 → ✅

3. 减数分裂过程中，每个四分体具有（） 

   A. 4个着丝粒 → 两个着丝粒

   B. 2对染色体 → 四对染色体

   C. 4个DNA分子 → ✅

   D. 2条姐妹染色单体 → 四对姐妹染色单体

4. 下列不属于减数第一次分裂主要特征的是（） 

   A. 同源染色体配对—联会 → ✅

   B. 着丝粒分裂 → 减I后期

   C. 四分体中非姐妹染色单体发生交叉互换 → ✅

   D. 同源染色体分离，分别移向细胞两极 → ✅

5. 下在减数分裂过程中，同源染色体的非姐妹染色单体之间发生交叉和互换的时期是（ ）

   A. 精细胞形成精子的时期 → 减数分裂完成

   B. 次级精母细胞产生精子细胞的时期 → 减II后期

   C. 精原细胞形成初级精母细胞时期 → 间期

   D. 初级卵母细胞产生第一极体的时期 → ✅

6. 某哺乳动物一个初级精母细胞经过减数命裂产生了染色体数目全部不正常的4个配子，最可能的原因是（）

   A. 一对同源染色体没有发生分离 → ✅

   B. 四分体时非姐妹染色单体交叉互换 → 不影响配子数目

   C. 一个次级精母细胞分裂不正常 → 有正常的

   D. 一条染色体的姐妹染色单体没有分开 → 有正常的

---

[^1]:[同源染色体] 一条来自父方，一条来自母方，形态、大小一般相同， 在减数分裂过程中配对的两条染色体。
[^2]:[联会] 减数分裂I前期，同源染色体两两配对的现象。
[^3]:[四分体] 指联会的一对同源染色体，包含四条染色单体(DNA)，叫做四分体。
[^4]:[交叉互换] 四分体时期，同源染色体的非姐妹染色单体之间，发生部分片段交换的现象。
